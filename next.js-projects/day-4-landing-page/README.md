# Day 4 Landing Page

## Project Overview

This is a responsive landing page built for Week 1 Day 4 of the internship roadmap. The page was created using Next.js and Tailwind CSS with AI assistance, then manually reviewed and cleaned.

## Landing Page Topic

AI Developer Assistant

## Features

* Responsive navbar
* Hero section
* Feature cards
* Workflow section
* Call-to-action section
* Footer
* Mobile-friendly layout

## Tech Stack

* Next.js
* JavaScript
* Tailwind CSS

## Run Locally

1. Open the project folder:

cd next.js-projects/day-4-landing-page

2. Install dependencies:

npm install

3. Start the development server:

npm run dev

4. Open in browser:

http://localhost:3000
